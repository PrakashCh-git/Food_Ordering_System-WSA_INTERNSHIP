Food Ordering System 

A full-stack Food Ordering System web application developed during my internship at Web Stack Academy.
This project allows users to browse food items, place orders, manage carts, and experience an online food ordering workflow.

Features:
User Authentication & Authorization
Browse Food Menu
Add to Cart Functionality
Place Orders
Responsive User Interface
Admin Panel for Managing Food Items
Order Management System
Secure Backend APIs

Backend
